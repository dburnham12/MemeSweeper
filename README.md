# MemeSweeper

Memesweeper builds on the traditional Minesweeper formula by keeping the original mine-based gameplay intact and layering meme-driven visuals and humor on top. The game emphasizes clean implementation of classic puzzle mechanics while enhancing player engagement through modern internet culture.

### MemeSweeper - Leaderboard Screens

Each game mode has its own leaderboard! When a player achieves a high score they will be added to the leaderboard in order based off of time. Lowest times are at the top and highest times are at the bottom.

### MemeSweeper - Profile

The game allows players to input their personal information as well as a username. The username is the most important feature of this system as it will be what the leaderboard uses to indicate high scores
